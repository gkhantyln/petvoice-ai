# Changelog

Bu dosya, PetVoice AI projesindeki tüm önemli değişiklikleri içerir.

Format [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) standardına dayanır ve bu proje [Semantic Versioning](https://semver.org/spec/v2.0.0.html) kullanır.

## [1.0.0] - 2026-03-11

### 🎉 İlk Sürüm

#### Eklenenler
- Google Gemini AI ile ses analizi
- İnteraktif spektrogram görselleştirme
- Tarayıcıdan ses kaydı özelliği
- Çoklu format desteği (WAV, MP3, OGG, FLAC)
- Kullanıcı kimlik doğrulama sistemi
- Evcil hayvan profil yönetimi
- Analiz geçmişi ve raporlama
- Admin paneli
- Responsive web tasarımı
- Türkçe dil desteği
- Rate limiting ve güvenlik özellikleri
- Kullanıcı dostu hata mesajları
- API anahtarı doğrulama
- Önbellekleme sistemi

#### Teknik Özellikler
- Flask 3.0+ web framework
- SQLAlchemy ORM
- Bootstrap 5 UI
- Librosa ses işleme
- Matplotlib görselleştirme
- SQLite/PostgreSQL desteği
- Docker desteği
- Comprehensive test suite

#### Dokümantasyon
- Detaylı README.md
- Kurulum rehberi
- Kullanım kılavuzu
- API dokümantasyonu
- Katkıda bulunma rehberi
- Lisans bilgileri

### Güvenlik
- CSRF koruması
- XSS koruması
- Güvenli şifre hashleme
- Input validasyonu
- Rate limiting
